# Support

## How to file issues and get help

This project uses [GitHub Issues](https://github.com/microsoft/aipm/issues) to track bugs and feature requests. Please search the existing issues before filing new issues to avoid duplicates. For new issues, file your bug or feature request as a new Issue.

For general questions and discussion about AIPM, please use [GitHub Discussions](https://github.com/microsoft/aipm/discussions).

## Microsoft Support Policy

Support for this project is limited to the resources listed above. This is an open-source project maintained by the AI Dev Tooling team and is not covered by any Microsoft paid support plan.
